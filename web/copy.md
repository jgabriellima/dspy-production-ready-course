## Sobre o Curso

### O que você vai aprender:

→ **Domine a arte de construir agentes de IA inteligentes**
  - Aprenda a criar agentes autônomos e inteligentes
  - Técnicas avançadas de prompt engineering
  - Arquitetura e design de sistemas de IA

→ **Programação declarativa e modular com DSPy**
  - Framework moderno para desenvolvimento de LLM applications
  - Componentes reutilizáveis e escaláveis
  - Best practices e padrões de projeto

→ **Do conceito à produção de forma profissional**
  - Workflow completo de desenvolvimento
  - Deploy e monitoramento em produção
  - Otimização e fine-tuning
