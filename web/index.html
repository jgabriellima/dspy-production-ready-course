Currículo

Do Fundamento à Otimização Avançada

7 módulos hands-on que transformam teoria em produção, com exercícios reais e métricas de validação

01

Fundamentos de DSPy e ReAct

Entenda a filosofia DSPy de programação declarativa, a arquitetura ReAct (Reasoning + Acting) e como LLMs podem ser usados como componentes modulares

Programação Declarativa ReAct Pattern LLM Orchestration
Você vai dominar: Configurar ambiente, conectar LLMs (OpenAI/Groq), criar suas primeiras Signatures
02

Arquitetura de Agentes com Tools

Aprenda a projetar agentes que interagem com o mundo real através de ferramentas. Crie data models robustos com Pydantic e implemente tool functions type-safe

$ def book_flight(user: str, flight_id: str) -> dict:
"""Ferramenta que reserva voos para usuários"""
Projeto prático: Sistema de atendimento ao cliente para companhia aérea
03

Construindo Seu Primeiro Agente

Implemente um agente ReAct completo que raciocina sobre qual ferramenta usar e quando. Domine o loop de Thought → Action → Observation que torna agentes realmente inteligentes

$ agent = dspy.ReAct(
signature=CustomerServiceSignature,
tools=[book_flight, cancel_booking, get_user_info]
)
Resultado: Agente funcional que resolve tarefas multi-step complexas
04

Avaliação e Métricas de Agentes

Aprenda a medir o que realmente importa: agentes não são apenas accuracy! Crie métricas multi-objetivo que avaliam completion, qualidade de resposta, eficiência e uso correto de tools

Task Success Rate Tool Accuracy Response Quality Efficiency
Habilidade crítica: Criar datasets de avaliação e baselines de performance
05

Otimização Automática: BootstrapFewShot

Veja a mágica acontecer! Use BootstrapFewShot para o DSPy automaticamente gerar e selecionar os melhores exemplos few-shot que maximizam suas métricas. Ganhos de 15-30% sem escrever prompts

$ optimizer = dspy.BootstrapFewShot(metric=multi_objective)
$ optimized = optimizer.compile(agent, trainset=examples)
⚡ Score: 0.548 → 0.714 (+30.3%)
Wow moment: Veja seu agente melhorar automaticamente em minutos
06

MIPRO: Otimização de Nível Expert

Domine a técnica mais avançada de DSPy. MIPRO otimiza instruções, exemplos E pesos simultaneamente usando programação matemática. Perfeito para squeeze the last 10% de performance

Instruction Tuning Example Selection Bayesian Search
Nível avançado: Técnicas usadas em agentes de produção de empresas como Microsoft e Google
07

Persistência e Versionamento de Modelos

Aprenda best practices de MLOps para agentes: serialização, versionamento, rollback, A/B testing. Nunca mais repita otimizações caras - salve e reutilize seus agentes otimizados

$ agent.save('models/v2.0-optimized.json')
# Em produção:
$ agent = dspy.load('models/v2.0-optimized.json')
Production-ready: Gerencie múltiplas versões e faça deploys com confiança
08

Deploy e Observabilidade em Produção

Coloque seu agente em produção com segurança. Aprenda monitoramento com Langfuse, métricas em tempo real, debugging de agentes, e otimização contínua baseada em dados reais

Langfuse Tracing Real-time Metrics Continuous Optimization
Resultado final: Agente otimizado, monitorado e pronto para escalar
